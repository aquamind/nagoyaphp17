## 雛形

### 1. インストール

```bash
$ git clone git@github.com:ttskch/nagoyaphp17.git
$ cd nagoyaphp17
$ composer install
```

### 2. 実装

[Nagoyaphp\Dokaku17\Dokaku17::run()](/src/Dokaku17.php#L9) を適切に実装する。

### 3. テスト

```bash
$ composer test
```

[Nagoyaphp\Dokaku17\Dokaku17Test::test()](/tests/Dokaku17Test.php#L24) のテストが全てパスすればOK。
